# Промпт для AI-агента

Ты работаешь над проектом **Сеанс** — статичной браузерной ежедневной игрой для русскоязычной аудитории.

Перед началом работы прочитай все документы:

- `docs/01_PRODUCT_BRIEF.md`
- `docs/02_GAME_RULES.md`
- `docs/03_DATA_MODEL.md`
- `docs/04_DATA_PIPELINE.md`
- `docs/05_UI_UX_BRIEF.md`
- `docs/06_TECH_ARCHITECTURE.md`
- `docs/07_ACCEPTANCE_CHECKLIST.md`
- `data/*.json`

Собери MVP на Vite + React + TypeScript.

Цель первой версии: полностью рабочая игра с локальным JSON-датасетом, двумя режимами `Фильмы` и `Сериалы`, фильтром периода, ежедневным выбором тайтла по московской дате, поиском, 10 попытками, плитками-подсказками, экраном победы/поражения, архивом, localStorage-статистикой и шарингом результата.

Не добавляй backend, авторизацию, админку, оплату, рекламу, аналитику, PWA, звуки и пользовательские загадки.

Работай итерациями:

1. Сначала создай структуру проекта и типы данных.
2. Затем подключи mock JSON и выбор загадки дня.
3. Затем реализуй поиск и игровую логику.
4. Затем собери UI.
5. Затем добавь статистику, архив и шаринг.
6. Затем проверь mobile-first адаптацию.

После каждой крупной итерации проверяй проект через `npm run build` и исправляй ошибки TypeScript.

Не меняй продуктовые решения из документов без явной причины. Если встречается неясность, выбирай самый простой вариант, который сохраняет механику игры.
