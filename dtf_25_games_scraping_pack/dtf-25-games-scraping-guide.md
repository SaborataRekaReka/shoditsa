# DTF: корпус обсуждений по 25 играм

Документ предназначен для скрапинг-агента. Его задача — собрать сырой корпус постов и комментариев, а не заранее определить «отношение DTF» к игре.

## Рекомендуемый пилот
**Starfield.** По игре есть отдельный тег, крупные редакционные ветки, пользовательские мнения, посты о модах и материалы поздней переоценки. Это позволяет проверить разные фазы и типы источников на одном объекте.

## Общий контракт выгрузки

- Сначала минимум 30 кандидатов, затем 8–12 выбранных постов.
- Приоритет постам с 100+ комментариями; если таких мало — топ по комментариям и содержательности.
- До 100 комментариев на пост по четырём корзинам: рейтинговые, ранние корневые, длинные ветки, несогласные/низко оценённые.
- Выход: `manifest.json`, `posts.jsonl`, `comments.jsonl` в отдельной папке игры.
- Только публичные страницы DTF; без обхода авторизации, без профилей и личных данных.
- Не делать выводов и не писать подсказки на этапе скрапинга.
